- `05-development-rules.md`：开发规范与统一约定
- `06-staging-rules.md`：测试环境规则

- `12-workflow-template-rules.md`：流程模板规则

## 文档维护规则

- 只记录长期有效规则，不记录单次过程。
- 不记录排查步骤、实现步骤、单次验证流水。
- 同类内容优先更新现有规则文档，不新增过程型文档。
- 默认使用中文维护。
